##前端规范

###文件规范
- 文件分类：
	- 基础类：base.css（标签的重置和设置默认值，常用的功能样式f-）
	- 公用类：common.css（“统一调用背景图和清除浮动或其他需统一处理的长样式”、“网站通用布局”、“通用模块和其扩展”、“元件和其扩展”）
	- 特殊类：page.css（当页面有特殊需求）
	- 皮肤类：skin.css（皮肤样式）

---

###注释规范
- 多行注释

	    /*
	     * 模块注释
	     * 模块注释
	     */  
	    .main{ width: 100px; }


- 单行注释

	    /* 星号左右空一格，这里是注释 */
	    .main{ width: 100px; }

- 行内注释

	    .main{ width: 100px; /* 这里是行内注释 */}
